# Models

A model adapter turns a provider's API into the framework's streaming interface.

Built-in adapters:

- Anthropic: `Anthropic`
- OpenAI: `OpenAI`
- Google Gemini: `Gemini`
- Moonshot: `Moonshot`
- OpenAI-compatible: `OpenAICompatible`

```ts
import { Agent } from 'roboport';
import { Anthropic } from 'roboport/models';

const agent = new Agent({
  model: new Anthropic('claude-opus-4-8', { thinking: 'medium' }),
  // …
});
```

`OpenAICompatible` targets any OpenAI-shaped endpoint via `baseUrl`. Thinking levels are unified across providers — `off | minimal | low | medium | high | xhigh` — and each adapter maps them onto its own wire format.

## Custom model

Extend `Model` and implement `streamMessage`; the non-streaming `createMessage` is derived from it for free. `searchWeb` backs the `web_search` tool — return `[]` if the provider has none.

```ts
import {
  Model,
  type CreateMessageParams,
  type ModelStreamEvent,
  type SearchHit,
} from 'roboport';

class EchoModel extends Model {
  async *streamMessage(
    params: CreateMessageParams,
  ): AsyncIterable<ModelStreamEvent> {
    const last = params.messages.at(-1);
    const text = typeof last?.content === 'string' ? last.content : '';
    yield { type: 'text-delta', text };
    yield { type: 'text-end', text };
    yield {
      type: 'message-end',
      id: 'echo-1',
      stopReason: 'end_turn',
      usage: { inputTokens: 0, outputTokens: 0 },
    };
  }

  async searchWeb(): Promise<SearchHit[]> {
    return [];
  }
}
```
