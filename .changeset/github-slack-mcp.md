Add `Github` and `Slack` MCP client presets to `roboport/mcp`. `Github` wraps the hosted GitHub MCP server with a personal access token (with optional `toolsets` and `readOnly` scoping); `Slack` exposes a curated set of Slack Web API tools (post message, reply in thread, list channels, channel history, users, reactions) via a bot token.
